# 🛠️ Cyber Toolkit Lab (SOC Analyst Toolkit)

A custom-built portable cybersecurity toolkit designed to simulate real-world SOC analyst workflows.

---

## 🔥 Overview
This project centralizes essential cybersecurity tools into a structured environment with a command-line launcher for efficient investigation and triage.

---

## 🧰 Tools

### 🌐 Networking
- Nmap (CLI)
- Wireshark
- Fiddler

### 🛡️ Pentesting
- Burp Suite
- OWASP ZAP (manual)

### ⚙️ Utilities
- Process Explorer
- Process Monitor
- TCPView
- Autoruns
- Strings
- Everything

### 🔍 Forensics
- KAPE (manual)
- Procmon

### 📧 Analysis / Decode
- CyberChef

---

## ⚡ Features
- Custom `.bat` launcher for quick tool access
- Organized investigation workflow structure
- Integrated cheat sheets and knowledge base
- Dedicated folders for logs, PCAPs, payloads, and malware

---

## 🧠 Example Workflow
1. Run Nmap scan
2. Capture traffic in Wireshark
3. Analyze processes with Process Explorer
4. Check persistence using Autoruns
5. Decode suspicious data with CyberChef

---

## 🎯 Purpose
Demonstrates:
- SOC analyst workflow understanding
- Real-world tool usage
- Incident triage methodology
